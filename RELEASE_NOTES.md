# Release 更新日志

## 0.2.0 - 播放器界面与歌词体验重构

这一版把 ECHO iPhone 从“手机遥控器”继续推进到更像真正音乐播放器的形态。

### 新增

- 播放页新增歌词模式：
  - 封面移动到左上。
  - ECHO 连接状态显示在封面下方。
  - 歌名与歌曲 tag 显示在封面右侧。
  - 中间显示可滚动歌词。
  - 当前歌词白色发光，前后歌词和时间戳为灰色。
  - 支持 LRC 时间戳解析。
  - 点击带时间戳的歌词可跳转播放进度。
- 歌词模式底部控制区：
  - 第一行左侧为播放进度。
  - 第一行右侧为可展开音量滑条。
  - 第二行为上一首、播放/暂停、下一首。
- 支持左右滑动切换播放、曲库、连接页面。
- 主要按钮和 dock 增加 `expo-blur` 高斯玻璃质感。
- README 新增中文默认首页、英文切换页、应用图标和完整预览图。

### 优化

- 关闭歌词后恢复居中封面的普通播放器样式，并带过渡动效。
- 页面切换从简单淡入升级为横向滑入。
- dock 与播放控制按钮的层次更接近 iOS 液态玻璃视觉。
- 音量调节在歌词模式中更紧凑，不再占用大面积空间。

### 修复

- 连接异常、曲库异常、手机播放异常改为弹窗提示，并避免相同错误重复弹出。
- EchoLink 断开时状态显示为红色“连接已断开”。
- 封面加载改为完整 URL 解析与原生图片兜底，减少空白和闪烁。

### 验证

- `npm.cmd run typecheck`
- `npx.cmd expo export --platform ios --output-dir build\export-check`

- 播放页、曲库页、连接页。
- 播放/暂停、上一首、下一首、进度拖动、音量拖动。
