# Release 更新日志

## 0.5.0 - SwiftUI 播放器、本地曲库、流媒体与真 DSP

发布日期：2026-07-15

这一版将 ECHO iPhone 整理为可独立使用的音乐播放器。手机本地曲库、ECHO、网易云音乐、原生 DSP、歌词、歌单和设置持久化现在共用一套 SwiftUI 播放体验。

### 播放与曲库

- 新增本地音乐模块，支持导入、扫描、读取元数据、读取本地 LRC、删除本地歌曲和统计本地音乐占用空间。
- 曲库页可在全部、ECHO、本地和流媒体之间切换。
- 本地曲库支持歌曲、专辑、艺术家、格式、收藏、最近播放等视图。
- 新增全局搜索，可检索 ECHO 与本地曲库。
- 新增可编辑歌单，支持创建、重命名、删除、收藏、置顶、添加歌曲和调整顺序。
- 播放页输出模式改为本地、流媒体、控制、串流四种模式。
- 本地播放支持本地队列、上一首、下一首、播放 / 暂停、单曲循环、进度拖动和音量控制。
- 原生播放列表支持当前歌曲标记、播放、移除、排序和清空，也会显示当前歌单。
- Now Playing 副标题按当前模式显示为 `Local Mode`、`Streaming Mode` 或 `Controlling Mode`。

### 流媒体

- 连接页加入网易云音乐入口，可选直连网易云 Web 接口或自托管 API 服务。
- 支持二维码登录、头像和昵称、用户歌单、歌曲搜索及播放。
- 网易云登录凭据保存在 iOS Keychain，接口地址、收藏与置顶状态保存在 App 本地。
- 歌单与歌曲列表支持完整分页获取，界面按需展示，避免只读取前 20 首。
- 当前流媒体歌曲已经开始播放后，即使登录源暂时不可用，暂停、seek 和音量仍可继续使用。

### DSP 与音频

- 新增 iOS 原生 DSP 模块 `echo-audio-dsp`，使用 `AVAudioEngine`、`AVAudioPlayerNode`、`AVAudioUnitEQ` 和动态处理器。
- 本地播放和串流播放优先走 DSP 链路，EQ、响度归一化、音量和 seek 会真实作用到音频播放。
- 新增 EQ 预设：均衡、低频、人声、清晰、暖声、夜间。
- 播放页新增 EQ 按钮和 EQ 面板，设置页也可以修改 EQ。
- 新增响度归一化设置，默认关闭。
- 音频 tag 支持选择展示项：输出模式、来源、串流能力、格式 / 音质、码率和时长。
- 支持展示 ECHO 端返回的专业音频字段：`codec`、`sampleRate`、`bitDepth`、`bitrate`。

### 连接与错误提示

- 新增 ECHO 连接总开关，默认关闭；关闭时不会自动连接、轮询或弹出连接异常。
- 连接页标题简化为“连接”。
- 连接页支持在 ECHO 与流媒体账号之间切换。
- 手动连接和配对连接继续支持 EchoLink Host、Port、Token。
- 支持相机扫描配对二维码，也支持从相册图片读取二维码。
- EchoLink Host、Port、Token 会保存到本地，不需要每次重新粘贴配对链接。
- 连接、曲库和本地播放错误改为弹窗提示，同类错误只提示一次，减少操作打断。
- 连接状态文案统一，“连接已断开”改为“ECHO未连接”。

### 设置与持久化

- 设置页重构为分组列表，点击列表项后展开对应设置。
- 语言设置接入全局文案，支持中文 / English。
- 新增默认页面、默认曲库、默认本地曲库视图等设置。
- 新增外源数据设置分组，默认关闭。LRCLIB、LrcAPI 和网易云音乐可单独开启。
- 新增“每次选择 / 自动匹配”、封面动态背景和网易云“直连 Web 接口 / 自托管”设置，并保存在 App 本地。
- 设置项保存到 App 本地个人数据，包括语言、默认页面、默认曲库、音频 tag、EQ、响度归一化、ECHO 连接开关和外源数据开关。
- 本地音乐状态会继续保存，包括收藏、最近播放和本地播放队列。

### 外源歌词与封面

- LRCLIB 用于优先获取歌曲歌词。
- LrcAPI 可补充歌词、封面和艺术家信息。
- 网易云音乐作为中文曲库补充，主要补充封面，也可作为歌词 fallback。
- 外源数据支持本地播放、串流播放和控制模式，并以歌曲名 + 艺术家作为缓存键。
- 歌词优先级为本地 LRC / EchoLink 歌词优先，外源歌词补位；外源之间优先 LRCLIB，其次网易云音乐。
- 封面优先使用本地文件或 EchoLink artwork URL。
- 如果 EchoLink 没有返回封面，或封面图片加载失败，App 会尝试检索网易云音乐封面。
- 外源返回一个或多个候选时都会打开原生选择弹窗；歌词、艺术家和封面可以分别指定来源，也可以选择“不使用”。
- 每个外源最多保留 2 首匹配歌曲，选择界面最多展示 6 个候选，不再只返回第一首。
- 曲库当前显示的无封面歌曲会顺序尝试外源补图，刷新曲库会重试未命中的歌曲。
- 新封面加载成功前会保留上一张稳定封面，减少空白和闪动。

### UI 与交互

- 播放、歌词、曲库、搜索、连接和设置页面迁移到 SwiftUI。
- 底部使用原生 TabView；支持系统 Liquid Glass，旧系统使用材质回退。
- 播放页整合封面、歌曲信息、tag、进度、主控按钮、音量、EQ、歌词和播放列表。
- 页面切换加入滑动和渐变动效。
- 音量展开条加长，并显示当前百分比。
- 歌词页支持自动滚动、高亮当前歌词和点击有时间戳的歌词 seek。
- 曲库 / 连接 / 设置页顶部移除多余的 `ECHO iPhone` 小标题。
- 曲库页标题下面的说明文字已移除，页面更简洁。
- 创建歌单改为紧凑小弹窗，外源选择改为半屏可滚动弹窗，主题背景会填满整个 sheet。
- 去除按钮的系统边框与自定义玻璃叠加，修复方形阴影和按下后玻璃合并遮挡。
- 修复搜索页顶部空白与标题下沉；锁定播放器布局尺寸，避免封面背景把进度、音量和控制按钮异常放大或遮挡。
- 封面背景缩放与模糊降低，保留轮廓但不干扰前景内容；EQ 右侧新增外源数据刷新按钮。

### 稳定性与日常使用

- 手动编辑连接信息时不再边输入边重连；只有保存后才切换连接。
- 旧连接和旧播放请求不会覆盖当前歌曲、进度或连接状态。
- 切换电脑播放时先等待 ECHO 确认成功；失败时保留手机正在播放的歌曲和正确输出模式。
- 修复状态轮询拉回正在拖动的进度条，以及进度在相邻秒数之间反复跳动的问题。
- 修复多来源弹窗中途切歌后，旧歌曲永久停留在“加载中”的问题。
- 本地曲库扫描失败时不会再把收藏、最近播放、歌单或队列当作无效数据清空。
- 设置、曲库状态和连接信息保存失败时会明确提示。
- 配对链接和持久化连接增加 Host、Token 与 `1-65535` 端口校验。
- SwiftUI 发出的原生操作已与 React 状态层逐项核对，没有未处理按钮。

### 已知限制

- LRCLIB 和网易云音乐外源数据需要手机能访问外网。
- 网易云流媒体可直连非官方 Web 接口，也可使用自托管 NeteaseCloudMusicApi；直连接口可能受上游变更影响。
- 网易云与外源元数据依赖非官方接口，稳定性取决于上游服务。
- Windows 环境可以验证 TypeScript 和 Expo 导出，但原生 Swift 模块仍需在 macOS / Xcode 或 GitHub Actions 中完成最终编译验证。

### 验证

- `npm.cmd run typecheck`
- `git diff --check`
- `npx.cmd expo config --type public`
- `npx.cmd expo export --platform ios --output-dir .expo-export-ux-review`

本次验证已通过 TypeScript 检查、Expo 配置解析、iOS Hermes bundle 导出和补丁格式检查。电脑端 EchoLink 测试因本地桌面源码未安装 `vitest` 依赖而未运行。

## 0.4.0 - 音乐播放器重构、设置页与专业音频 Tag

发布日期：2026-06-23

这一版把 ECHO iPhone 从早期控制界面继续推进成更完整的音乐播放器，重点处理播放页拥挤、组件遮挡、专业音频信息展示和基础设置入口。

### 新增

- 新增设置页，并加入底部 dock。
- 设置页支持界面语言选择：中文 / English。
- 设置页支持选择音频 tag 展示项：输出模式、来源、串流能力、格式音质、码率、时长。
- 曲库新增筛选：全部 / 可串流 / 本地。
- 连接页新增 Host、曲库数量、可串流数量概览。

### 音频信息

- 支持展示 ECHO 桌面端发送的专业音频字段：`codec`、`sampleRate`、`bitDepth`、`bitrate`。
- 音频质量 tag 合并为更易读的格式，例如 `FLAC 48kHz/24bit`。
- 兼容 `48000` 和 `48` 两种采样率输入，避免显示异常。

### 播放体验

- 播放页整体重构为深色玻璃音乐播放器风格。
- 音量控制改为辅助控制行里的可展开滑条，减少常驻占位。
- 上一首 / 下一首改成双箭头，降低与页面滑动或返回手势的视觉冲突。
- 歌词页保留大字号、高对比、自动滚动与点击歌词跳转。
- 播放列表弹窗保持在播放页上方展示，并修复暗色背景下标题对比。

### 布局与稳定性

- 收紧封面、主控按钮、tag 与控制区尺寸，降低小屏重叠风险。
- 歌词页或音量展开时允许滚动，避免 dock 遮挡控制区域。
- 连接页 Host 文本增加单行截断，避免撑破布局。
- 保持手机端改动独立，没有修改 PC 端源码。

### 验证

- `npm.cmd run typecheck`
- `git diff --check`
- `npx.cmd expo export --platform ios --output-dir build\export-check`

### 已知限制

- 语言设置目前先覆盖导航、页面标题/描述和设置页文案，播放页内部分功能文案仍保持中文。
- 设置项持久化已在 0.5.0 补上。
